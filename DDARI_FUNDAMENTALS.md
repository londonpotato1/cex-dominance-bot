## 1. 따리의 핵심 원리

### 1.1 기본 개념
```
따리 = 신규상장 + 보따리
해외에서 코인 매수 → 국내 상장 시 매도 → 가격 차이 수익
```

### 1.2 김프(김치프리미엄) 발생 원리
- **국내 가격 > 해외 가격** = 김프 발생
- **매수세 > 매도세** = 가격 상승 = 김프 확대
- **매도세 > 매수세** = 가격 하락 = 역프 발생

### 1.3 김프 계산 공식
```
김프(%) = (국내가격 - 해외달러가격×환율) / (해외달러가격×환율) × 100
```

### 1.4 따리가 가능한 이유
1. 거래소별로 **다른 가격** 형성
2. 코인은 **거래소 간 전송 가능** (블록체인 기술)
3. 국내거래소는 **내국인만 사용 가능** → 폐쇄시장

## 2. GO/NO-GO 판단 데이터

### 2.1 🎯 핵심 판단 공식 (가격 없이도 분석 가능!)
```
흥따리 확률 = f(입금액 예측, 거래량 예측)

핵심: 거래량 > 입금액 → 흥따리
      입금액 > 거래량 → 망따리
```

### 2.2 입금액 예측 데이터 (공급 측면)

| 데이터 | 확인 방법 | 영향 |
|--------|----------|------|
| **거래소 핫월렛 물량** | arkm.com, 이더스캔 | 적을수록 흥따리 ↑ |
| **네트워크 속도** | 체인 explorer | 느릴수록 흥따리 ↑ |
| **브릿지 난이도** | 브릿지 사이트 확인 | 어려울수록 흥따리 ↑ |
| **현선갭** | datamaxiplus.com | 클수록 헷징 어려움 → 입금액 ↓ |
| **에어드랍 물량** | 토크노믹스 분석 | 적을수록 흥따리 ↑ |
| **DEX 유동성** | defillama.com | 적을수록 흥따리 ↑ |
| **최고거래소 출금 정책** | 거래소별 확인 | 막혀있으면 흥따리 ↑ |

### 2.3 거래량 예측 데이터 (수요 측면)

| 데이터 | 확인 방법 | 영향 |
|--------|----------|------|
| **시장 상황 (시황)** | 비트코인 가격, 전체 거래량 | 불장일수록 거래량 ↑ |
| **직전 상장 결과** | 상장 복기 | 흥따리 연속 가능성 |
| **상장 시간대** | 거래소 공지 | 저녁(17-21시) best |
| **프로젝트 관심도** | SNS, 뉴스 | 높을수록 거래량 ↑ |
| **업비트 기대감** | 빗썸 선상장 시 | 있으면 거래량 ↑ |

### 2.4 토크노믹스 분석 (TGE 상장용)

| 항목 | 확인 포인트 | 흥따리 유리 조건 |
|------|------------|------------------|
| **펀드레이징** | 투자자 목록, 투자금액 | 유명 VC 참여 |
| **초기유통량** | 전체 대비 % | 낮을수록 유리 |
| **에어드랍 비중** | 전체 대비 % | 높으면 초기 매도압력 |
| **시가총액** | FDV, MC | 낮을수록 유리 (25M 이하 초저시총) |
| **장외거래가(OTC)** | OTC 마켓 | 현재가 대비 할인율 |

### 2.5 시가총액 기준 (참고용)

| 구분 | 시가총액 | 특징 |
|------|---------|------|
| 초저시총 | 25M 이하 | 운전 가능성 ↑, 변동성 ↑ |
| 저시총 | 25M~60M | 후펌핑 가능성 |
| 중시총 | 60M~150M | 일반적 |
| 고시총 | 150M~300M | 안정적, 알파 ↓ |
| 초고시총 | 300M 이상 | 망따리 확률 ↑ |

## 3. 흥따리/망따리 판정 기준

### 3.1 손바뀜 비율 (핵심 지표)
```
손바뀜 비율 = 5분 거래량 / 입금액

- 비율 > 5: 대흥따리 (예: AVAIL 19배)
- 비율 > 2: 흥따리
- 비율 ≈ 1: 무난
- 비율 < 1: 망따리
```

### 3.2 흥따리 조건 체크리스트

**✅ 공급 측면 (입금액 ↓)**
- [ ] 거래소 핫월렛 물량 적음
- [ ] 네트워크 속도 느림 (POW, 자체 메인넷)
- [ ] 브릿지 어려움/불가
- [ ] DEX 유동성 적음
- [ ] 에어드랍 클레임 어려움
- [ ] 해외거래소 출금 막힘
- [ ] 현선갭 큼 → 헷징 어려움

**✅ 수요 측면 (거래량 ↑)**
- [ ] 불장 (업비트 일거래량 10조+)
- [ ] 직전 상장 흥행
- [ ] 프로젝트 관심도 높음 (김치코인, 트럼프 등)
- [ ] 상장 시간 황금대 (17-21시)
- [ ] 업비트 기대감 (빗썸 선상장 시)

### 3.3 망따리 신호

**⚠️ 경고 신호**
- 입금액 대비 거래량 부족
- 선반영 심함 (상장 전 이미 고점)
- 따리꾼 참여도 과다 → 세력의 유동성 출구로 활용
- 현선갭 0%에 가까움 (헷징 쉬움)
- 솔라나 체인 + 장세 안좋음

### 3.4 빗썸 vs 업비트 특성

| 항목 | 빗썸 | 업비트 |
|------|------|--------|
| 유동성 | 10억대 | 100억대 |
| 상장기준 | 유연함 | 까다로움 |
| 최소 입금액 | 20억 (25년 이후) | 50억 이상 |
| 흥따리 조건 | 입금액 20억 + 거래량 100억+ | 입금액 100억 이하 + 거래량 500억+ |
| 사이클 | 황썸→빗썸→망썸 (3주 주기) | 더 안정적 |

## 4. 상장 유형별 전략

### 4.1 TGE 상장 (Token Genesis Event)

**특징**
- 해외거래소 출금 보통 막혀있음 (24시간 후 개시)
- 헷징 어려움 (선물페어 없거나 현선갭 큼)
- 벨류에이션 어려움 (첫 가격 형성)
- **가장 높은 알파 기회**

**전략**
1. **물량 확보**: DEX, 에어드랍, OTC
2. **벨류에이션**: 펀드레이징, 토크노믹스, 유사 코인 비교
3. **출금 타이밍**: 바이낸스/게이트 출금 시점 파악
4. **상장 연기 대비**: 익일 오전 상장 시나리오

**주의사항**
- TGE 직후는 사실상 매매 (헷징 불가)
- 손실폭 관리 필수
- 분할매수 원칙

### 4.2 직상장 (기존 코인 원화 상장)

**특징**
- 해외에서 이미 거래 중인 코인
- 헷징 쉬움
- 입금액 많을 확률 높음
- 상대적으로 낮은 알파

**전략**
1. **현선갭 확인**: datamaxiplus.com
2. **핫월렛 물량 확인**: arkm.com
3. **시황 확인**: 불장이면 고시총도 김프 가능

### 4.3 옆상장 (BTC마켓 → 원화마켓)

**특징**
- 업비트 내 기존물량 존재
- 빠른 상장 (20분~1시간)
- 거래량 적게 나옴
- **미리 공부 가능** (예상문제)

**전략**
1. **사토시마켓 종목 리스트 파악**
2. **핫월렛 물량 미리 조사**
3. **빗썸 견제상장 가능성 체크**
4. **암살 물량 준비** (미리 입금)

## 5. 헷지 전략

### 5.1 선물 헷징 (현선갭 잡기)

```
현물 매수 (롱) + 선물 매도 (숏) = 가격 변동 제거

기준: 갯수 동일 (금액 아님!)
```

**비용 요소**
- 현선갭 손해 (현물 > 선물 일반적)
- 거래 수수료
- 펀딩피 (8시간/4시간마다)

**수익 요소**
- 김프 수익
- 현선갭 축소 시 수익

### 5.2 마진 헷징 (대출로 현선갭 회피)

```
담보 예치 → 코인 대출 → 국내 매도 → 상황 따라 상환/매수
```

**장점**
- 현선갭 손해 없음
- 펀딩피 부담 없음
- 거래시간 단축

**단점**
- 이자 발생 (최근 상승 추세)
- 대출 물량 경쟁 치열
- 청산 리스크 (마진콜)

**대출 플랫폼**
- CEX: 바이낸스 론, 바이비트 론, 게이트 론
- DEX: AAVE, Compound

### 5.3 현선갭 전략 요약

| 상황 | 행동 | 결과 |
|------|------|------|
| 현선갭 작음 → 커짐 | 현선 잡고 풀기 | 현선갭 수익 |
| 현선갭 큼 (음펀비) | 빌려서 먹기 | 현선갭 + 펀비 수익 |
| TGE 직후 | 헷징 어려움 | 생따리 or 패스 |

## 6. 과거 사례 교훈

### 6.1 대박 사례 분석

| 코인 | 시기 | 김프 | 핵심 원인 |
|------|------|------|----------|
| **MINA** | 23년 | 200%+ | 체인 터짐 + 자체 메인넷 |
| **CKB** | 24년 | 300%+ | POW + 비트겟만 출금 |
| **MANTA** | 24.1 | 2,000% | TGE + 체인 터짐 + 30명 입금 |
| **AVAIL** | 24.7 | 1,000%+ | DEX 없음 + OTC만 + 4억 입금 |
| **MOVE** | 24.12 | 100%+ | 불장 + 게이트만 출금 + 400억 입금 |
| **ME** | 24.12 | 90% | 불장 + MOVE 후속 |
| **MIRA** | 25.9 | 100%+ | TGE + 클레임 어려움 |

### 6.2 초대박 공통점
1. **체인 문제**: 자체 메인넷, POW, 네트워크 과부하
2. **입출금 제한**: 출금 막힘, DEX 없음
3. **후따리 어려움**: 브릿지 어려움, 물량 부족

### 6.3 실패 사례 교훈

| 코인 | 교훈 |
|------|------|
| **EVER** | 마이그레이션 리스크, 재단 규모 확인 필수 |
| **XPL** | 하이퍼리퀴드 선선갭 청산 리스크 |
| **VANA** | 선반영 심함 → 따리꾼이 유동성 출구 |
| **MOCA** | 업비트 쏠림 → 빗썸 빈집 대흥따리 |

### 6.4 불장→망장 사이클 교훈

```
불장 초입 → 흥따리 연속 (MOVE, ME)
↓
따리꾼 참여 급증 → 입금액 증가
↓  
선반영 심화 → 따리꾼이 출구
↓
망따리 연속 (VANA, SONIC, ANIME)
↓
상장 열기 죽음 → 거래량 감소
↓
다음 사이클 대기
```

## 7. 위험 관리 원칙

### 7.1 절대 원칙

1. **올인 금지**: 감당 가능한 손실폭 설정
2. **리스크 분석 필수**: 왜 RBN은 되고 EVER는 안됐나?
3. **'절대'는 없다**: 모든 전략에 예외 존재
4. **복기 필수**: 매 상장마다 기록, 분석

### 7.2 리스크 체크리스트

**⚠️ 고위험 상황**
- [ ] 재단 규모 작음 (백커 확인)
- [ ] 마이그레이션/리브랜딩 예정
- [ ] 선선갭 아비트라지 (청산 리스크)
- [ ] 노헷징 생따리
- [ ] 프리마켓 숏 (mark price 이슈)

**✅ 저위험 상황**
- [ ] 현선갭 작음 + 헷징 완료
- [ ] 마진 대출 성공
- [ ] 메이저 재단 (Paradigm, a16z 등)
- [ ] 솔라나/이더 메인 체인

### 7.3 포지션 사이징

```
불장 + 흥따리 시그널: 큰 베팅 (현선갭 5%까지 감수)
평시: 중간 베팅 (현선갭 2%까지)
망장: 소액 베팅 or 패스 (현선갭 1% 이하만)
```

### 7.4 손절 기준

- **TGE 생따리**: -20% 도달 시 손절 고려
- **현선 헷징**: 펀딩피 손실 > 기대 김프 시 청산
- **마진 헷징**: 청산 위험 1.3 이하 시 추가 담보

- 시황: 불장 / 보통 / 망장

- 선물페어: 있음 / 없음

### GO/NO-GO 판정
- [ ] 입금액 < 거래량 예상
- [ ] 손바뀜 비율 > 2 예상
- [ ] 헷지 가능 or 손실 감당 가능
