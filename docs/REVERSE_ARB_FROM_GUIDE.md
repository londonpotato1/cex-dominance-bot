## 📝 역프 아비트라지 전략 정리

### 전략 1: 국내 매수 + 해외 숏 (역따리)

```
1. 조건: 국내 역프 발생 (국내 < 해외)
2. 실행:
   - 국내 현물 매수 (업비트/빗썸)
   - 해외 선물 숏 (바이낸스/바이빗) - 가격 헷징
3. 전송:
   - 국내 → 해외 코인 전송
4. 청산:
   - 해외 현물 매도
   - 숏 포지션 청산
5. 수익 = 역프% - 수수료 - 현선갭
```

### 전략 2: 현현갭 아비트라지

```
1. 조건: 역프 + 출금 제한 (곧 해제 예상)
2. 실행:
   - 역프 거래소에서 현물 매수
   - 출금 해제 대기
3. 청산:
   - 해제 후 다른 거래소로 전송
   - 높은 가격에 매도
```
