# DataMaxi+ vs gap_dashboard_v3 비교 분석

_작성일: 2026-01-31_

## 1. DataMaxi+ 기능 정리

### 1.1 주요 기능

| 기능 | 설명 | 상세 |
|------|------|------|
| **아비트라지 탐색기** | 여러 전략별 프리미엄 기회 탐색 | Spot-Spot, Spot-Perp, Perp-Perp |
| **상장 아비트라지** | 신규 상장 김프 기회 감지 | 온체인 입금 상태 추적 포함 |
| **펀딩비 대시보드** | 거래소별 펀딩비 비교 | 로그인 필요 (유료?) |
| **알림 시스템** | 조건 맞춤 자동 알림 | Telegram, 이메일, 브라우저 푸시 |
| **거래소 공지** | 주요 거래소 공지 모니터링 | 실시간 업데이트 |
| **텔레그램 채널 큐레이션** | 암호화폐 커뮤니티 트렌드 | 채널별 메시지 수집 |

### 1.2 지원 거래소

**DataMaxi+ 공식 발표**: 20+ 거래소, 7,000+ 토큰, 200,000+ 포트폴리오 조합

**확인된 거래소** (Arbitrage Explorer 기준):
- **한국**: 업비트, 빗썸
- **해외**: Binance, Bybit, OKX, Gate.io, Bitget, MEXC, KuCoin, Huobi 등
- **DEX**: 다수 (구체적 목록 미확인)

### 1.3 아비트라지 탐색기 상세 기능

```
📊 필터 옵션:
├── Position Size (포지션 크기)
├── Exchanges Used (사용 거래소)
├── Crypto Owned (보유 코인)
├── Minimum Premium (최소 프리미엄)
├── Premium Duration (프리미엄 유지 시간)
├── Network Selection (네트워크 선택)
├── Withdrawal Fee (출금 수수료)
├── PnL (예상 손익)
└── Fee Deduction (수수료 차감)

🎯 전략별 필터:
├── Spot-Spot (현물 간 아비트라지)
├── Spot-Perp (현선 아비트라지)
└── Perp-Perp (선물 간 아비트라지)

📍 지역 필터:
├── All (전체)
├── Overseas to Korea (해외→한국)
└── Overseas Only (해외만)
```

### 1.4 상장 아비트라지 기능

**핵심 특징**:
- ✅ 거래소 신규 상장 실시간 감지
- ✅ 거래 시작 전 온체인 입금 상태 확인 (업비트, 빗썸)
- ✅ 과거 상장 사례 및 입금 기록 히스토리
- ✅ 가격, 입출금 상태 업데이트

### 1.5 펀딩비 대시보드 (로그인 필요)

**예상 기능** (대시보드 힌트 기반):
- 거래소별 펀딩비 비교
- 펀딩 간격 (funding intervals)
- 정산 시간 (next settlement time)
- 히스토리 데이터
- 인덱스 가격, 최저/최고 비율

### 1.6 알림 시스템

**알림 채널**:
- Telegram
- 이메일
- 브라우저 푸시

**알림 트리거** (예상):
- 특정 프리미엄 도달 시
- 신규 상장 감지 시
- 펀딩비 급변 시
- 각 전략 페이지에서 원클릭 알림 생성

## 2. gap_dashboard_v3 현재 기능

### 2.1 주요 기능

| 기능 | 상태 | 설명 |
|------|------|------|
| **GO/NO-GO 분석** | ✅ 구현 | 상장 전 진입 판단 |
| **실시간 김프** | ⚠️ 부분 | 프리미엄 계산 |
| **상장 감지** | ✅ 구현 | 업비트/빗썸 마켓 Diff + 공지 폴링 |
| **VC/MM 인텔리전스** | ✅ 구현 | VC, MM 정보 수집 |
| **TGE 언락 분석** | ✅ 구현 | 토큰 언락 리스크 |
| **핫월렛 추적** | ⚠️ 부분 | Etherscan 기반 |
| **펀딩비** | ✅ 신규 | Binance/Bybit 기본 |
| **텔레그램 봇** | ✅ 구현 | 명령어 기반 인터랙션 |
| **과거 상장 히스토리** | ✅ 구현 | CSV 로드 (67건) |

### 2.2 지원 거래소

**현물**:
- 업비트 (KRW)
- 빗썸 (KRW)

**선물** (펀딩비용):
- Binance Futures
- Bybit

### 2.3 현선갭 계산 방식

```python
# gap_dashboard_v3 - 가격 기반 계산
premium = (kr_price - global_price) / global_price * 100
```

**현재 방식**: 단순 가격 비교 (Last Price 기반)
**오더북 기반 아님**: 실제 체결 가능 가격이 아닌 마지막 체결가 사용

## 3. 비교표

| 항목 | DataMaxi+ | gap_dashboard_v3 | 비고 |
|------|-----------|------------------|------|
| **지원 거래소 수** | 20+ | 4개 (업빗, 빗썸, 바낸, 바빗) | DataMaxi+ 압도적 |
| **토큰 수** | 7,000+ | 업빗/빗썸 리스팅 기준 | DataMaxi+ 압도적 |
| **현선갭 계산** | 오더북 기반 (executable) | 가격 기반 (Last Price) | DataMaxi+ 더 정확 |
| **Spot-Spot 아비** | ✅ 지원 | ❌ 미지원 | 추가 필요 |
| **Spot-Perp 아비** | ✅ 지원 | ❌ 미지원 | 추가 필요 |
| **Perp-Perp 아비** | ✅ 지원 | ❌ 미지원 | 추가 필요 |
| **상장 감지** | ✅ 실시간 | ✅ 30-60초 폴링 | 유사 |
| **온체인 입금 상태** | ✅ 업빗/빗썸 | ⚠️ 핫월렛 추적 | DataMaxi+ 더 직관적 |
| **DEX 통합** | ✅ 지원 | ⚠️ DexScreener 연동 | DataMaxi+ 통합적 |
| **펀딩비** | ✅ 로그인 필요 | ✅ 기본 (BTC, ETH) | gap_v3 무료 |
| **알림** | Telegram, Email, Push | Telegram only | DataMaxi+ 다양 |
| **GO/NO-GO** | ❓ 미확인 | ✅ 자동 판단 | gap_v3 차별화 |
| **VC/MM 분석** | ❓ 미확인 | ✅ 구현 | gap_v3 차별화 |
| **가격** | 유료 추정 | 무료 | 비용 차이 |

### 3.1 DataMaxi+ 강점

1. **거래소 커버리지**: 20+ 거래소로 더 많은 기회 포착
2. **오더북 기반 계산**: 실제 체결 가능한 프리미엄 표시
3. **다양한 전략**: Spot-Spot, Spot-Perp, Perp-Perp 모두 지원
4. **온체인 입금 추적**: 상장 전 입금 상태 직관적 표시
5. **필터 다양성**: 포지션 크기, 수수료, 네트워크 등 세밀한 필터
6. **알림 다채널**: Telegram + Email + Browser Push

### 3.2 gap_dashboard_v3 강점 (차별화 포인트)

1. **GO/NO-GO 자동 판단**: 진입 여부 의사결정 지원
2. **VC/MM 인텔리전스**: 프로젝트 배경 분석
3. **TGE 언락 분석**: 공급 리스크 예측
4. **상장 히스토리 학습**: 과거 패턴 기반 예측
5. **무료**: 비용 부담 없음
6. **커스터마이징**: 소스코드 접근 가능

## 6. 결론 및 권장사항

### 6.1 단기 액션 아이템 (이번 주)

1. **오더북 기반 프리미엄 계산 구현**
   - 업비트/빗썸 오더북 API 연동
   - Ask/Bid 기반 실제 체결 가능 프리미엄

2. **Binance/Bybit 현물 추가**
   - 해외→한국 아비트라지 기회 확대
   - 현선갭 계산 정확도 향상

3. **수수료 자동 차감 계산**
   - 출금 수수료, 거래 수수료 반영
   - 실질 PnL 표시

### 6.2 중기 목표 (1개월)

1. **다전략 지원**: Spot-Spot, Spot-Perp
2. **네트워크 속도 정보 통합**: 체인별 평균 확정 시간
3. **알림 조건 다양화**: 프리미엄 %, 지속 시간, 거래량 조건

### 6.3 gap_dashboard_v3 차별화 전략

DataMaxi+와 직접 경쟁하기보다 **차별화된 가치** 제공:

1. **무료 + 오픈소스**: 누구나 사용, 커스터마이징 가능
2. **GO/NO-GO 의사결정**: 진입 판단 자동화 (DataMaxi+에 없음)
3. **프로젝트 분석**: VC/MM, TGE 언락, 리스크 스코어링
4. **한국 시장 특화**: 업비트/빗썸 상장 아비트라지 전문
5. **학습 기능**: 과거 상장 히스토리 기반 패턴 분석
